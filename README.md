# 🚀 AP Webcraft Portfolio

A modern, responsive, and SEO-optimized portfolio website showcasing software development projects, final-year academic projects, and custom web solutions.

## 🌐 Live Demo
https://apwebcraft22.netlify.app/


---

## 📌 About

AP Webcraft is a portfolio website designed to showcase professional software projects, web applications, AI solutions, cloud-based applications, and academic projects.

The website focuses on providing a premium user experience with modern UI, responsive design, project search, filtering, testimonials, FAQs, and SEO optimization.

## ⚙ Installation

Clone the repository

```bash

```

```bash

```

Run locally

Simply open:

```
index.html
```

or use Live Server in VS Code.
